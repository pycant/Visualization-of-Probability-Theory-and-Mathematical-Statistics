## 目标
- 在 Chapter3/5/8 提升炫酷程度：更强的霓虹光晕、渐变动画与玻璃拟态质感，同时保持可读性与性能。
- 保持结构与交互不变，仅在视觉层面增强，参考首页整体风格并加以升级。

## 配色与主题
- 使用双主色 + 点缀色：
  - Chapter3：紫 (violet) + 青 (cyan) + 点缀蓝 (sky)
  - Chapter5：橙 (amber) + 粉 (pink) + 点缀紫 (violet)
  - Chapter8：绿 (green) + 蓝绿 (teal) + 点缀青 (cyan)
- 将三色以 CSS 变量暴露：`--accent`, `--accent-2`, `--accent-ghost`, `--accent-glow`，用于背景、边框、阴影与动画。

## 背景层次（页面容器）
- 叠加 3 层背景：
  1) 顶部左右两侧径向霓虹光晕（`radial-gradient`，`mix-blend-mode: screen`）。
  2) 横向扫描光束（淡色 `linear-gradient`，`translateX` 周期动画）。
  3) 低密度星空点阵（多重 `box-shadow` 或 `background-image` 伪随机点，透明度极低）。
- 轻微动画：光束 10s 缓慢扫过；无硬核粒子，确保性能。

## 卡片与内容块
- 卡片（`bg-dark-card`）：
  - 双层阴影：内发光（浅色）+ 外投影（章节色），`hover` 时增强对比与轻微 `translateY(-2px)`。
  - 渐变边框：用 `border-image` 或伪元素实现 2px 渐变边框，保证 Tailwind 兼容。
- 内容交替：每页分区交替出现“浅色着色卡片”与“常规深色卡片”，通过 `accent-ghost` 渐变背景提升节奏。
- 标题与图标：
  - 标题渐变文本（章节主辅色过渡）与发光下划线（宽度动画）。
  - 圆形徽记图标带霓虹外圈与轻微脉冲（慢速 `pulse`）。

## 微交互与装饰
- 分区进入动画：卡片使用 `cardEnter` 分批入场（与导航下拉一致但更柔和）。
- 悬停涟漪：点击卡片添加半透明涟漪（复用已在导航中使用的样式）。
- 小标签（如“概述”、“核心要点”）：使用渐变胶囊标签，在标题右侧构成视觉锚点。

## 作用域样式实现
- 在每页 `<head>` 添加 `<style>`：
  - 定义章节主题变量与背景伪元素样式：`body > .min-h-screen::before/::after` 实现光晕与光束；另建 `::marker`/`background` 点阵层。
  - 卡片边框与阴影、标题渐变与下划线动画、胶囊标签样式、图标外圈发光。
  - 为 `text-neon-*` 提供兜底色，以便在无全局定义时仍呈现接近章节色调。
- 不新增全局 CSS 文件；仅对 `chapter3.html`、`chapter5.html`、`chapter8.html` 增量插入作用域样式。
