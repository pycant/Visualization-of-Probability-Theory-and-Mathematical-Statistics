# 可视化教学案例网站介绍

本文档面向课程使用者、维护者与贡献者，系统性介绍位于 `templates/` 目录中的整站页面结构、导航与交互设计、视觉与配色规范、主要教学内容页、静态资源与技术栈，以及后续迭代方向与使用指南。目标是帮助你在最短时间内理解网站的整体信息架构与实现思路，并在需要时快速定位到相应的页面进行编辑或扩展。

---

## 1. 项目概览

- 项目名称：可视化教学案例（面向概率论与数理统计相关课程的交互式网页教材）
- 主要目录：
  - `templates/`：网站的各类页面模板（教学章节、索引与专题页、登录与管理页、局部组件）。
  - `templates/partials/`：可复用的页面局部（导航栏 `navbar.html`、页脚 `footer.html`）。
  - `static/`：静态资源（样式、脚本、图片、视频、第三方库）。
  - `docs/`：项目说明与课程文档（教学设计、场景案例、需求与任务记录、网站介绍等）。

站点以主题章节为主线，配合交互动画、图表与视频内容进行讲解。导航栏提供快速跳转、章节列表与页面选择器等能力，确保在不同屏幕尺寸下都能保持良好的可用性与信息密度。

## 2. 目录结构与页面类型

### 2.1 模板页面（`templates/`）

主要页面包括：
- `index.html`：网站首页或总览入口，通常用于引导用户进入重点章节与专题。
- 主题章节与概念页：
  - `probability_distributions.html`（概率分布）
  - `random_variables.html`（随机变量）
  - `expectation_variance.html`（期望与方差）
  - `law_of_large_numbers.html`（大数定律）
  - `hypothesis_testing.html`（假设检验）
  - `interval_estimation.html`（区间估计）
  - `appendix.html`（附录与扩展）
- 章节型页面：
  - `chapter1.html`、`chapter3.html`、`chapter5.html`、`chapter8.html`（示例章节，体现不同主题的内容组织与交互风格）。其中 `chapter8.html` 标题为“方差分析与回归分析”。
- 其他辅助与试验页：
  - `flip_card_test.html`、`test.html`（交互组件实验或教学原型验证）。
- 登录与管理：
  - `login.php`、`admin_subscribers.php`、`create_admin.php`、`submit_email.php`（简单的后台或订阅管理相关页面）。

### 2.2 局部组件（`templates/partials/`）

- `navbar.html`：全局导航栏，提供品牌区、首页入口、章节切换、页面链接与页面选择器等功能。
- `footer.html`：全局页脚，提供版权信息、相关链接与反馈渠道。

通过在各页面中引入或动态包含这些局部组件，可以保证导航与页脚的一致性，减少维护成本。

### 2.3 静态资源（`static/`）

- `static/css/`：站点样式。示例：`random_variables.css`、`卡方分布.css` 等专页样式。
- `static/js/`：站点脚本。示例：
  - `include-navbar.js`、`include-footer.js`：动态引入局部组件，统一导航与页脚。
  - `chapter1.js`、`toolbox.js`、`url-click-handler.js` 等交互逻辑脚本。
- `static/img/`：图标与插画，如 `machine-learning.svg`、`medical-diagnosis.svg`、视频封面等。
- `static/libs/`：第三方库，如 `chart/`、`gsap/`、`katex/`、`mathjax/`、`three/` 等，支持图表绘制、动画、数学公式、3D 可视化等能力。
- `static/videos/`：教学视频资源，如 `正态分布.mp4`、`卡方分布.mp4`、`随机变量.mov` 等。

## 3. 导航与信息架构（`partials/navbar.html`）

导航栏承担了网站的全局定位与快速跳转职责，包含下列构成与策略：

- 品牌与首页入口：
  - `.brand` 与 `#nav-home-link` 提供最小宽度与不换行约束，保证在缩窄布局下仍清晰可见。
- 章节列表与切换按钮：
  - `#chapters-toggle` 与 `#chapters-list` 使用卡片式展示，每个章节项拥有独立的主色与次色，并在标题、图标与左侧高亮条中保持一致。
  - 动效统一使用章节色：悬停光晕、掠过闪光、点击涟漪均会根据章节的自定义颜色变量自动适配。
- 页面链接区（`#nav-page-links`）：
  - 采用“单行横向滚动”的策略：`overflow-x: auto`、`flex-wrap: nowrap`、`white-space: nowrap`，避免挤占其它导航元素。
  - 宽高限制与滚动条样式统一，确保链接较多时仍可操作、不会换行或遮挡。
- 页面选择器（`#nav-page-select`）：
  - 通过 `width: clamp(140px, 18vw, 220px)` 结合 `min/max-width` 与 `flex: 0 0 auto`，保证控件处于可视区域，避免被压缩或推出视口。
- 左侧分组间距统一：
  - 针对 `#global-nav > div > div > div.flex.items-center.space-x-4` 的子元素（`a`/`button`/`span`）设置一致的 `gap`，并清理左右外边距，防止双重间距。

总体目标是在保证信息密度的同时，避免换行导致的布局抖动，并在不同屏幕宽度下保持操作舒适度与可读性。

## 4. 视觉风格与配色系统

站点采用“章节主题色”理念，每个章节卡片定义一组 CSS 变量：

- 主色与次色：`--accent`、`--accent-2`
- 派生透明色：`--accent-shadow`、`--accent-2-shadow`（光晕阴影），`--accent-ghost`（掠过闪光），`--accent-ripple`（点击涟漪），`--accent-bar-shadow`（左侧高亮条阴影），`--accent-icon-glow`（图标发光）

这些变量确保在悬停、点击、掠过等交互中，光效、阴影与辉光统一呈现为该章节的主题色，形成一致的感知与记忆锚点。标题与图标采用相同的配色逻辑，左侧霓虹条进一步强化章节识别度。

字体方面，每个章节项可绑定不同的字体族与颜色，以区分内容类型或强调页面语气。整站在字号、行高与对比度上遵循可读性优先原则，保证在课堂投屏与个人设备上都有良好呈现。

## 5. 主要页面与教学内容概述

以下为 `templates/` 目录中部分关键页面的功能与内容概览：

- `index.html`：入口与导览
  - 介绍网站的学习路径、核心模块与最新更新；引导用户进入重点章节或专题页。

- `probability_distributions.html`：概率分布
  - 汇总离散与连续分布，配合图表与交互控件展示形状、参数与典型应用。
  - 可配合视频与示例数据演示参数变化对分布的影响。

- `random_variables.html`：随机变量
  - 介绍随机变量的定义、性质与常见操作；通过交互组件帮助理解抽样与变换。

- `expectation_variance.html`：期望与方差
  - 从直观图示解释期望与方差的含义，结合练习与图表加深理解。

- `law_of_large_numbers.html`：大数定律
  - 通过模拟与动画展示样本均值的收敛行为；配合可视化更直观地呈现理论结论。

- `hypothesis_testing.html`：假设检验
  - 介绍假设设定、检验统计量、显著性水平与 p 值等概念；支持交互式计算与判定流程演练。

- `interval_estimation.html`：区间估计
  - 说明区间估计的定义、置信水平与构造方法；通过图示帮助理解区间的覆盖概率。

- `appendix.html`：附录
  - 汇总数学公式、符号约定与参考资料；作为学习与备查的工具页。

- `chapter1.html`、`chapter3.html`、`chapter5.html`、`chapter8.html`
  - 示例章节，体现统一导航、色彩与交互规范；其中 `chapter8.html` 的核心主题为“方差分析与回归分析”。页面可包含图表、交互滑块、案例分析与视频片段。

- `flip_card_test.html` / `test.html`
  - 用于组件原型试验与交互行为验证，帮助在上线前评估体验与性能。

- `login.php`、`admin_subscribers.php`、`create_admin.php`、`submit_email.php`
  - 简单的登录与订阅管理入口，支持课程运营或通知机制。

## 6. 交互设计与动画策略

- 悬停与点击反馈：
  - 章节卡片在悬停时出现同色系光晕与阴影，掠过时产生柔和闪光，点击后生成同色涟漪，强调操作的可感知性与愉悦感。

- 链接区域滚动：
  - 当页面链接较多时，`#nav-page-links` 维持单行横向滚动，不会挤占其它导航控件，用户可用滚轮或触控进行横向浏览。

- 页面选择器适配：
  - `#nav-page-select` 通过宽度钳制与不收缩策略，确保始终处于可视区域并保持单行显示。

- 统一间距与对齐：
  - 左侧功能组子元素使用一致的 `gap` 控制，避免因类名叠加导致的双重间距或不规则分布。

所有交互均围绕“稳定布局、清晰反馈、主题一致”三个原则进行设计，以支持课堂演示与个人自学两种主要场景。

## 7. 响应式布局与适配

- 不换行策略：
  - 对品牌区、首页入口、章节切换按钮与页面选择器设置最小宽度与不换行，避免在窄屏下出现断行或遮挡。

- 可滚动策略：
  - 对链接集合采用横向滚动替代换行，保留整体导航结构不变的同时提升内容承载能力。

- 条件显示：
  - 某些控件（如选择器）可根据断点进行显示与隐藏，以保持导航在不同屏幕上都易读易用。

## 8. 静态资源与第三方库

- 图表与可视化：`static/libs/chart/`、`three/` 等，用于绘制统计图形与 3D 演示。
- 动画与交互：`gsap/` 提升动效表现，配合 CSS 变量实现主题一致的光效。
- 数学排版：`katex/`、`mathjax/` 支持公式排版与课堂讲解。
- 媒体素材：`static/img/` 与 `static/videos/` 存放插画与教学视频，配合页面内容进行案例与演示。

资源管理遵循“就近使用、按需加载”的原则，避免过度依赖与冗余引入，确保页面加载性能与课堂展示稳定性。

## 9. 可访问性与可用性原则

- 对比度与字号：保证在投屏与移动端上可清晰阅读；避免过度装饰影响信息获取。
- 键盘与触控：交互元素尽可能支持键盘导航与触控滚动，兼顾不同操作习惯。
- 一致的反馈：颜色与光效统一映射到章节主题，用户可快速建立“颜色—内容”的认知关联。

## 10. 内容组织与教学设计

- 理论—案例—练习：以理论知识为骨架，配合真实或模拟数据案例，并提供交互练习巩固理解。
- 文档支撑：`docs/design/` 与各主题文档记录课程脚本、案例与交互构想，为页面实现提供素材与依据。
- 跨页一致性：通过统一导航与视觉规范，降低跨页切换的认知负担，使章节间衔接顺畅。

---

## 11. 性能优化与工程实践

- 样式与脚本分层：公共样式与脚本按模块化管理，页面级样式仅包含该页所需内容。
- 资源复用：局部组件（导航、页脚）统一维护，减少重复与差异导致的维护成本。
- 可扩展的配色变量：章节主题色通过 CSS 变量集中管理，新增章节只需补充变量即可继承交互光效。

---

## 12. 使用指南

1. 进入首页（或任一章节页）后，利用顶部导航进行章节与页面切换。
2. 若页面链接较多，可在 `#nav-page-links` 中进行横向滚动浏览，链接文本保持单行显示。
3. 在章节列表中悬停或点击，观察与该章节主题色一致的光效反馈，快速定位章节。
4. 若需要按页面名称快速跳转，可使用 `#nav-page-select` 选择器进行筛选与切换。
5. 在教学演示中建议使用桌面端浏览器，以获得最佳的信息密度与交互体验。

## 13. 常见问题（FAQ）

- 为什么导航不换行而是横向滚动？
  - 为避免布局抖动与元素遮挡，横向滚动可在保持结构稳定的前提下承载更多链接。

- 章节颜色与光效如何统一？
  - 每个章节卡片定义一组主/次色与派生透明色变量，悬停、掠过与点击动效均从这些变量生成，确保一致性。

- 小屏设备如何适配？
  - 通过最小宽度钳制、不换行策略与条件显示，保留核心操作入口，同时减轻拥挤与遮挡。

## 14. 迭代建议与扩展方向

- 章节主题扩展：为更多章节配置主题色与图标，保持识别度与沉浸感。
- 交互强度调节：根据课堂反馈适当增强或减弱光效强度与透明度，提升舒适度。
- 媒体内容增强：增加交互式实验与数据可视化案例，强化“看—想—做”的闭环。
- 文档联动：在每个页面中增加“查看文档来源”入口，直达 `docs/` 中相关说明与脚本。

本网站以“统一导航、统一色彩、统一交互”为基础，为概率论与数理统计类课程提供可视化与交互式学习体验。通过在 `templates/` 与 `static/` 中维持清晰的结构与规范，我们能够在保证稳定性的同时不断扩展新的章节与内容。若你计划新增页面、调整配色或扩展交互，请参考本文档的结构说明与规范建议进行实施，以保持整体风格的一致与演进的可控。
