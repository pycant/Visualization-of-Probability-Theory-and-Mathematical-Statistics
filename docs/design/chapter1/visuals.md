# 第一章 可视化方案设计（分小节）

说明：为每个小节设计对应的可视化或交互演示，统一使用 `VisCard` 组件样式；下方为建议实现与交互点。

## 1.1 随机事件及其运算
- 1.1.1 随机现象：
  - 可视化：掷骰/投币模拟器，动态柱状图显示频率；实时更新期望频率与偏差。
  - 交互：开始/暂停、试验次数滑块、显示均值线。
- 1.1.2 样本空间：
  - 可视化：结果枚举网格（如两枚骰子相加的有序/无序样本空间），支持切换有序/无序视图。
  - 交互：高亮选中结果、统计子集大小。
- 1.1.3 随机事件：
  - 可视化：选定子集为事件，突出显示；展示 `A ⊆ Ω` 的关系。
  - 交互：勾选规则（例如和为偶数、点数≥4），动态统计事件概率（古典法）。
- 1.1.4 随机变量：
  - 可视化：从结果到实数映射（函数映射线/表）与对应直方图。
  - 交互：切换不同变量定义，观察分布形状变化。
- 1.1.5 事件间的关系：
  - 可视化：维恩图（A、B）可拖拽交叠，显示 `A∩B`、`A∪B`、`A\\B` 等区域比例。
  - 交互：滑块控制集合大小与重叠度，动态展示公式数值。
- 1.1.6 事件间的运算：
  - 可视化：集合运算动画（补集、差集），结合样本空间网格高亮。
  - 交互：按按钮演示运算效果与对应概率变化。
- 1.1.7 事件域：
  - 可视化：展示事件族闭包概念（有限例子演示），高亮满足闭包的组合。
  - 交互：选择若干基事件，自动生成可见事件域的子集组合列表。
- 习题1.1：
  - 题型：选择/判断/匹配（事件与样本空间、关系与运算）。
  - 交互：即时反馈与解析，可开启“提示”。

## 1.2 概率的定义及其确定方法
- 1.2.1 公理化定义：
  - 可视化：性质说明卡（非负性、规范化、可加性），配合小型集合图演示。
  - 交互：点选性质查看示例与反例动画。
- 1.2.2 排列与组合：
  - 可视化：排列组合计数器（n、k 滑块），同时显示 `n!`、`A^n_k`、`C^n_k` 与排列树示意。
  - 交互：参数调整、展开树、导出枚举列表。
- 1.2.3 频率方法：
  - 可视化：重复试验频率收敛曲线（大样本近似），与理论概率对比。
  - 交互：设置样本量与事件条件，实时曲线更新。
- 1.2.4 古典方法：
  - 可视化：等可能模型下的概率＝有利数/总数，动态计数展示。
  - 交互：改变约束条件（例如投掷两骰和的条件），自动计数并显示概率。
- 1.2.5 几何方法：
  - 可视化：区域面积比（如点落在圆环区域），用画布随机点模拟与解析面积比对比。
  - 交互：拖拽改变几何参数，切换蒙特卡罗/解析法。
- 1.2.6 主观方法：
  - 可视化：先验分布与主观概率示例（简单场景选择题），展示不同先验下的判断差异。
  - 交互：选择“专家A/B”的主观概率，对比决策结果差异。
- 习题1.2：计数与概率确定方法综合。

## 1.3 概率的性质
- 1.3.1 可加性：
  - 可视化：互斥事件并集概率加法，维恩图标注数值与公式。
  - 交互：切换是否互斥，观察加法公式变化。
- 1.3.2 单调性：
  - 可视化：`A ⊆ B ⇒ P(A) ≤ P(B)` 演示；显示示例计数与公式。
  - 交互：拖拽改变包含关系，实时数值更新。
- 1.3.3 加法公式：
  - 可视化：`P(A∪B) = P(A)+P(B)-P(A∩B)` 动画演示。
  - 交互：控制 `P(A∩B)` 大小，查看数值平衡。
- 1.3.4 连续性：
  - 可视化：递增/递减事件列的极限概率动画（简化模型）。
  - 交互：步长与终止条件设置，显示收敛示意。
- 习题1.3：性质应用题。

## 1.4 条件概率
- 1.4.1 定义：
  - 可视化：条件限制后样本空间缩小的示意图；树状结构标注条件分支。
  - 交互：选择条件，更新 `P(A|B)` 的数值与解释。
- 1.4.2 乘法公式：
  - 可视化：事件路径乘积（树图），路径概率乘积=联合概率。
  - 交互：展开/折叠路径，查看每一步的条件概率。
- 1.4.3 全概率公式：
  - 可视化：分割样本空间的一组互斥完备事件，累加得到目标事件的概率。
  - 交互：编辑分割与权重，验证公式。
- 1.4.4 贝叶斯公式：
  - 可视化：前后验变换条形图；频率表直观对照。
  - 交互：修改先验与似然，自动更新后验；显示公式分解。
- 习题1.4：条件概率与贝叶斯综合题。

## 1.5 独立性
- 1.5.1 两个事件的独立性：
  - 可视化：`P(A∩B)=P(A)P(B)` 的数值对比与误差提示。
  - 交互：参数化 `P(A)`、`P(B)`，展示是否独立的判断。
- 1.5.2 多个事件的相互独立性：
  - 可视化：三事件独立/两两独立的区别示例（维恩图+数值表）。
  - 交互：切换不同独立关系配置，观察条件约束。
- 1.5.3 试验的独立性：
  - 可视化：重复试验的独立性与相关性对比（示例过程）。
  - 交互：打开/关闭相关性，观察概率变化。
- 习题1.5：独立性判断与构造题。
