# 网站内容深度概述

注：网站模板位于 `templates` 目录，静态资源位于 `static` 目录；首页入口为 `templates/index.html`。

---

## 1. 首页（index.html）

- 内容概览：
  - 首页以“核心概念”模块为主轴，呈现章节卡片（随机事件与概率、随机变量、概率分布、期望与方差、多维随机变量、大数定律、统计量及其分布、区间估计、假设检验、方差分析与回归分析、附表）。
  - 每个卡片包含主题图标、简短介绍与“进入章节/深入了解”的链接，部分卡片提供微缩预览（通过 iframe 缩放预览对应页面）。
  - Section 列表与效果：
    - 章节卡片区（`随机事件与概率`、`随机变量`、`概率分布`、`期望与方差`、`多维随机变量`、`大数定律`、`统计量及其分布`、`方差分析与回归分析`、`附表`）：概念卡片悬浮倾斜与光效反馈；点击进入章节或锚点；部分卡片带微缩预览（iframe）。
    - 实践应用区（`洪水频率分析与极值分布`、`军事分析：寻找遗失的炮弹`、`游戏策略：末地之门定位`、`假设检验：英雄平衡与农业评估`、`直播打赏异常检测`）：每项为案例卡片，点击可跳转到相关教学页面或专题说明，体现跨学科应用场景。
    - 工具与测试区（`LaTeX渲染测试`、`Markdown表格测试`）：用于技术验证与展示公式/表格渲染效果，评委可快速判断排版与可读性。
    - 页脚分区（`快速链接`、`学习资源`、`联系我们`）：统一的导航与资源入口，方便评委在首页完成全站跳转。
- 核心交互：
  - 概念卡片悬浮与点击：悬浮呈现轻微倾斜与光效，点击直接跳转到目标页面或锚点（如 `hypothesis_testing.html#practical-applications`）。
  - 微缩预览（iframe）：部分卡片内嵌缩放预览，鼠标悬停保持预览，点击进入对应页面；在长列表下方预览不会阻塞整体滚动。
  - 章节卡片的“进入章节/深入了解”按钮：点击后导航到 `templates` 内相应文件；按钮具备点击涟漪反馈，提升操作确定性。
  - 页面内概念预览（class `concept-preview`）：以 `<a>` 标签实现，包含目标 `href`，点击立即跳转，适合评委快速串联跨页内容。
  - 页眉交互（简述）：顶部导航动态高亮当前页面；`#nav-page-links` 单行水平滚动且不换行，`#nav-page-select` 下拉可直接切换到页内重要段落。
  - 目录侧边栏（TOC）粘性定位：长内容滚动时保持在可视区域，便于评委快速跳转。
- 学习目标：
  - 形成对全站知识结构的总览与学习路径的初步规划，按章节逐步进入。

## 2. 第 1 章：随机事件与概率（chapter1.html）

- 内容概览：
  - 页面以“第 1 章 随机事件与概率”为标题，分为 1.1 随机事件及其运算、1.2 概率的定义与确定方法（含频率法模拟与其他方法概览）、1.3 概率的性质、1.4 条件概率（含计算器）、1.5 独立性（含示例与练习）。
  - 左侧为章节目录（锚点跳转），右侧为分块内容卡片，含说明、公式、练习提示与交互组件。
  - Section 列表与效果：
    - 1.1 随机事件及其运算：介绍样本空间、事件关系与运算（并、交、补）；通过示例与图示帮助理解运算性质与德摩根律。
    - 1.2 概率的定义与确定方法：从公理化定义出发，结合频率法的掷骰子模拟，强调“频率趋近概率”的直觉效果。
    - 1.3 概率的性质：展示加法公式、乘法公式等基本性质，并辅以练习题引导巩固。
    - 1.4 条件概率：给出条件概率的定义与计算方式，配合交互式计算器实现即时计算与独立性提示。
    - 1.5 独立性：解释独立与互斥的区别，提供示例与判断练习，形成常见误区的纠正。
    - 交互演示区（掷骰子模拟）：动态累加频率，实时展示频率逼近理论概率的效果。
    - 交互工具区（条件概率计算器）：根据输入参数计算 P(A|B)，并辅助进行独立性粗判，体现“从公式到工具”的落地。
- 核心交互：
  - 掷骰子模拟（元素：`#dice-roll`, `#dice-reset`）：
    - 点击“掷一次”使用伪随机生成 1–6 点数，实时累加各面出现次数与总试验次数，显示相对频率曲线或表格；
    - 点击“重置”清空统计与界面结果，恢复初始状态；
    - 交互在快速连续点击下保持稳定，不会出现重复触发或无响应。
  - 条件概率计算器（元素：`#cond-calc`, `#pAB`, `#pB`, `#pA`）：
    - 在输入交集概率 P(A∩B) 与条件事件概率 P(B) 后，点击“计算”即时输出 P(A|B)=P(A∩B)/P(B)；
    - 若提供 P(A)，同时给出“与独立性的关系”提示：当 P(A∩B)≈P(A)P(B) 时提示可能独立；
    - 对非法输入（负数、>1、P(B)=0）给出错误提示或忽略计算，保护边界。
- 学习目标：
  - 掌握事件运算与概率公理；理解条件概率、全概率公式与贝叶斯公式的应用逻辑；区分互斥与独立。

## 3. 随机变量（random_variables.html）

- 内容概览：
  - 围绕“离散型与连续型随机变量”的定义、分布函数与密度函数、常见随机变量示例、变量变换与构造、图形化展示进行组织。
  - 页面通常包含对“取值集合、概率赋值/密度、期望与方差”的逐步讲解。
  - Section 列表与效果（当前版本示例性呈现）：
    - 随机数生成器：提供基础随机数演示，结合图示说明样本与分布之间的关系。
    - 定义与分类：离散与连续的区分，配合典型示例（几何/泊松/正态）建立直觉；
    - 分布函数与密度函数：通过曲线或条形图展示 F(x)、f(x)/p(x) 的含义与差异；
    - 数字特征：展示期望与方差等特征如何反映集中趋势与离散程度；
    - 变量变换与构造：示意常见变换（线性变换、函数变换）对分布与特征的影响。
- 核心交互：
  - 参数面板与图形联动：调整取值范围、分布参数（如 λ、p、μ、σ）后，曲线/条形图与数值特征（期望、方差）同步更新；
  - 离散/连续切换：在离散变量下显示概率质量函数（PMF）与条形图，在连续变量下显示概率密度函数（PDF）与光滑曲线；
  - 典型示例快速切换：内置若干示例（如几何/泊松/正态），一键切换以对比图形与数字特征差异。
- 学习目标：
  - 理解随机变量的本质与与其分布之间的关系；为后续“概率分布”“期望与方差”奠定基础。

## 4. 概率分布（probability_distributions.html）

- 内容概览：
  - 页面分为“理论知识深入探索”板块，以“多页翻转卡片”形式展开正态分布、泊松分布等的理论介绍、公式推导、期望与方差推导、性质与定理。
  - 正态分布卡片四页结构：分布介绍、概率密度函数推导（最大熵约束）、期望与方差推导、重要性质（68-95-99.7 规则、线性变换、可加性、标准化）。
  - 泊松分布卡片结构：分布介绍（计数型场景）、概率质量函数推导（小时间间隔假设 λdt、独立性、稀疏性）、应用场景举例。
  - Section 列表与效果（示例挑选）：
    - 正态分布理论：基本概念/历史/应用场景 → 概率密度函数推导（基本假设、推导过程、最终结果） → 数字特征推导（期望、方差、重要结论） → 重要性质（对称性与 68-95-99.7、线性变换、可加性、标准化）。通过翻卡逐页阅读，形成完整推导链条。
    - 泊松分布理论：基本概念/历史/应用场景 → 概率质量函数推导（假设、过程、结果） → 数字特征推导（期望、方差、重要结论） → 重要性质（可加性、二项极限、正态近似、矩母函数）。突出“计数型事件”的建模直觉。
    - 指数分布理论：基本概念/历史/应用场景 → 概率密度函数推导 → 数字特征推导（期望、二阶矩、方差） → 重要性质（无记忆性、最小值性质、与其他分布关系、矩母函数）。强调“等待时间”模型的核心性质。
    - 二项分布理论：基本概念/历史/应用场景 → PMF 推导（设定、过程、结果） → 数字特征 → 性质（伯努利推广、正态近似、泊松极限、可加性）。连接试验次数与成功概率的直觉。
    - 均匀分布理论：基本概念/历史/应用场景 → PDF 推导（假设、过程、结果） → 数字特征 → 性质（等概率性、线性变换、最大熵、随机数生成基础）。为随机抽样与仿真提供直觉。
    - 卡方分布理论：基本概念/数学定义/应用场景 → PDF 推导（基本推导、CDF、与伽马关系） → 数字特征 → 性质（可加、正态近似、统计应用）。为检验与方差估计提供支撑。
    - t 分布理论：基本概念/历史/应用场景 → PDF 推导（假设、过程、结果） → 数字特征（期望、方差、高阶矩） → 性质（渐近、对称、尾部、平方关系、样本均值分布）。解释小样本平均的分布特征。
    - F 分布理论：基本概念/历史/应用场景 → PDF 推导 → 数字特征（期望、方差、众数） → 性质（倒数性质、与 t 关系、渐近、分位数、方差比检验）。连接方差比较与方差分析。
    - 贝塔分布理论：基本概念/历史/应用场景 → PDF 推导 → 数字特征（期望、二阶矩、方差） → 性质（对称性、特殊情况、共轭先验、变换）。服务于贝叶斯与比例建模。
    - 伽马分布理论：基本概念/历史/应用场景 → PDF 推导 → 数字特征（期望、二阶矩、方差） → 性质（可加、特殊情况、共轭先验、尺度变换）。连接等待时间与总量模型。
    - 额外分布（威布尔、对数正态、帕累托、拉普拉斯、狄利克雷、逆伽马）：每个包含“定义与常用形式”“数字特征”“性质与应用”，用于扩展工程与经济场景。
    - 关系图谱与专题应用：
      - 概率分布关系图谱：展示极限/构造/特殊关系的可视化，形成分布间的认知地图；
      - 洪水频率分析与极值分布：围绕江西水利工程背景，说明分布选择、参数设定（Gumbel/GEV）、PDF、重现期与洪水位关系、重现期计算器、案例与案例详情。实现“理论—工程应用”的直接链接。
    - 页脚分区：`快速链接`、`学习资源`、`联系我们` 统一资源入口。
- 核心交互：
  - 多页翻卡（导航点与左右箭头）：点击导航点或箭头切换页面，当前页高亮；在过渡动画期间暂时禁用二次切换，防止状态错乱；
  - 公式与推导分步呈现：每页按“引入 → 约束 → 结论”顺序显示，评委可逐步跟随逻辑链；
  - 页面状态记忆：返回同一卡片时保留上次停留页，便于断点续读（若浏览器未清理状态）。
- 学习目标：
  - 掌握主流分布的数学表达与关键性质；理解建模假设与推导链条；能在应用场景中选择合适分布。

## 6. 大数定律（law_of_large_numbers.html）

- 内容概览：
  - 解释频率趋近概率的数学原理与实验直觉；常以模拟方式展示样本均值随样本量增大而收敛的趋势。
  - Section 列表与效果：
    - 大数定律的数学表述：用清晰的卡片说明弱/强大数定律的核心陈述，配合示例强调差异。
    - 中心极限定理的桥梁作用：解释 LLN 与 CLT 的关联与差异，说明为何样本均值的分布趋近正态。
    - 弱大数定律与强大数定律：并列呈现，辅以文字与图示加强比较理解。
    - 收敛速度与误差分析：通过模拟与图形展示随样本量增长的误差收敛趋势与波动范围。
    - 推广与变形：介绍在更一般条件下的 LLN 及其应用范围。
    - 页脚分区：`快速链接`、`学习资源`、`联系我们`。
- 核心交互：
  - 随机模拟控制：
    - 设置样本量 n（滑条/输入框），点击“开始/暂停”生成独立样本并计算样本均值；
    - 折线图显示均值随试验次数的轨迹，累计直方图展示样本均值分布；
    - 提供“重置”清空数据，确保不同试验可对比；
    - 在高频点击下保持队列化处理，避免图形渲染阻塞。
- 学习目标：
  - 理解“重复试验—统计规律性—频率逼近概率”的联系；为统计推断的可靠性提供理论支撑。

## 7. 区间估计（interval_estimation.html）

- 内容概览：
  - 讲解用样本统计量构造包含未知参数的区间（置信区间），强调“估计值与不确定性”共同呈现。
  - 典型包括均值/比例的区间估计，涉及正态、t、二项等分布的临界值。
  - Section 列表与效果（结构以表单与数轴可视化为主，部分标题可能非 h 标签）：
    - 概念与方法：统一阐述置信区间含义与常见构造思路。
    - 单样本均值区间：已知/未知总体方差两种情形分别说明（Z/t 方法），配合可视化数轴展示上下限。
    - 单样本比例区间：给出适用条件与计算流程，强调样本量与近似的影响。
    - 方差/方差比区间（如两样本方差比）：结合卡方/F 分布说明区间构造逻辑。
    - 方法选择提示：结合前提条件给出“选择 Z 或 t”的直观指引。
    - 页脚分区：`快速链接`、`学习资源`、`联系我们`。
- 核心交互：
  - 参数输入与即时计算：填写样本量、均值/比例、方差/标准差、置信水平，点击“计算”后显示区间上下限与解释文字；
  - 可视化区间：在数轴上标注估计值与左右端点，随参数变动平滑更新；
  - 方法选择提示：根据前提（是否已知总体方差、样本量大小、分布假设）提示选用 Z 或 t；
  - 学习进度解锁（若开启）：页面可能提供“理解程度”按钮，完成关键阅读/练习后解锁进阶内容；
  - 异常输入保护：对非法参数（负数、非数值、样本量过小）提示并阻止计算，以保证结论可靠。
- 学习目标：
  - 认识参数估计的不确定性；掌握区间估计在实际决策中的使用方法。

## 8. 假设检验（hypothesis_testing.html，开发中）

- 内容概览：
  - 系统讲解“提出原假设与备择假设—选择统计量与拒绝域—计算检验统计量—给出结论”的流程；覆盖常见检验（均值、比例、方差、两样本检验等）。
  - Section 列表与效果：

## 16. 页眉与导航（partials/navbar.html）与页脚（partials/footer.html）说明

- 内容概览：
  - 导航包含品牌区域、首页链接、章节切换按钮、页面链接集合（带水平滚动、单行显示）、页内页面选择器（`#nav-page-select`）。页脚包含版权、辅助链接与联系信息。
- 近期优化（重要）：
  - `#nav-page-links` 限宽（`max-width: 50vw`）与限高（单行 40px）并开启水平滚动；单行不换行；避免挤压其它元素。
  - 导航左侧分组统一子元素间距（a/button/span，gap: 12px），避免双重外边距造成不一致。
  - `#nav-page-select` 设定单行显示与宽度夹持（`clamp(140px, 18vw, 220px)`），防止被其他元素挤出视野。
- 核心交互：
  - 动态加载与高亮：导航在页面加载时自动注入链接集合，并根据当前路径高亮对应入口；
  - 链接区域（`#nav-page-links`）：内容超出时可水平滚动，链接保持单行不换行，避免遮挡下拉；
  - 页内选择器（`#nav-page-select`）：下拉选择后平滑滚动至对应段落，并在选择器中保持当前选中项；宽度夹持与不收缩策略避免在窄屏失效；
  - 外部点击关闭与位置修正：下拉展开后可点击其它区域关闭；在窗口尺寸变化时自动修正位置与宽度，保证可达性。
