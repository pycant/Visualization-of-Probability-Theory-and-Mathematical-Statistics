## 第二部分：理论基础回顾（4-5分钟）

### 【解说词】
首先，让我们回顾一下概率论的基础知识。在我们之前学过的分布中，正态分布、指数分布、伽马分布等都描述了随机变量的一般行为。但是，当我们关注的不是一般情况，而是极端情况时，比如一年中的最大降雨量、最高洪水位，这时候就需要用到极值分布。

**极值分布的生成机制**

想象一下，如果我们有365天的日降雨量数据，我们最关心的往往是这一年中的最大值。根据Fisher-Tippett-Gnedenko定理，当我们对这些最大值进行适当的标准化后，它们会收敛到三种极值分布之一：

1. **Gumbel分布**（极值I型）：当原始数据来自指数族分布时
2. **Fréchet分布**（极值II型）：当原始数据具有重尾特征时  
3. **Weibull分布**（极值III型）：当原始数据有有限上界时

**为什么降水极值服从极值分布？**

这里有深刻的物理和数学原理。降水的形成需要多个气象因子协同作用：水汽输送、抬升机制、凝结核浓度、温度梯度等。只有当这些因子同时达到极值状态时，才会产生极端降水。这种"多因子协同极值"的特征，正是极值分布理论的典型应用场景。

**4.1 分布类型选择与参数设置**

现在让我们进入交互式洪水频率分析工具。首先，我们需要选择合适的分布类型。

【操作演示】点击Gumbel分布选项

【操作演示】展示重现期图表

## 第六部分：理论深化与拓展（2-3分钟）

### 【画面】
- 显示数学公式和推导过程
- 展示与其他分布的关系图

### 【解说词】

通过刚才的实践操作，我们不仅学会了如何使用工具，更重要的是理解了极值分布的深层原理。

**理论拓展**

极值分布与我们学过的经典分布有着深刻的联系：
- 正态分布的最大值 → Gumbel分布
- 指数分布的最大值 → Gumbel分布  
- 均匀分布的最大值 → Weibull分布

这种联系不是偶然的，而是概率论中极限定理的体现。就像中心极限定理告诉我们样本均值的分布，极值理论告诉我们样本极值的分布。

**参数估计方法**

在实际应用中，我们需要从历史数据估计分布参数：
- 矩估计法：简单但可能不够精确
- 最大似然估计：理论最优但计算复杂
- 概率权重矩法：专门为极值分布设计，稳健性好
